## 👋 소개 (Introduction)

**Cherry**는 글로벌 K-POP 팬덤을 위한 굿즈 전문 C2C 거래 플랫폼입니다.  
우리는 범용 중고 거래 플랫폼의 비효율을 해결하고, 포토카드, 앨범, 한정판 굿즈 등 오직 팬들을 위한 안전하고 전문적인 거래 공간을 제공합니다.

우리의 미션은 **"아이돌 굿즈에 특화된 사용자 경험이 얼마나 의미 있는 거래 흐름을 만들어낼 수 있는가?"** 라는 가설을 검증하는 것입니다.

## 🚀 Why Cherry?

- **🔍 특화된 탐색 경험**: 냉장고, 자전거 사이에서 최애의 포카를 찾지 마세요. Cherry는 아이돌 굿즈에 최적화된 필터와 카테고리를 제공합니다.
- **✨ 명확한 정보**: 아티스트, 상품 종류, 희소성 등 정형화된 데이터를 통해 내가 사는 굿즈가 무엇인지 정확히 알 수 있습니다.
- **🤝 신뢰와 안전**: '찐팬'들이 모인 커뮤니티에서, 범용 장터보다 더 안전하고 매너 있는 거래를 경험하세요.

## ✨ MVP 핵심 기능 (Core Features)

- **아이돌 중심 탐색**: 아티스트별, 굿즈 종류별, 희소성(레어템)별 탐색
- **1:1 거래 채팅**: 구매자와 판매자 간의 매끄러운 소통
- **스마트한 상태 관리**: `판매중` → `예약중` → `판매완료`로 이어지는 거래 흐름 시각화
- **찜 (Pick)**: 갖고 싶은 '성배' 아이템을 저장하고 알림 받기
- **AI 글쓰기**: AI가 도와주는 간편하고 매력적인 상품 등록

## 🛠 기술 스택 (Tech Stack)

**Client (`cherry-web`)**
- **Core**: React, TypeScript, Vite
- **Styling**: Tailwind CSS (Custom Design System)
- **State**: Zustand
- **Architecture**: FSD (Feature-Sliced Design)

**Server (`cherry-server`)**
